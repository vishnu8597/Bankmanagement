Frequently Asked Questions

What is the minimum balance after opening an account?

All the customers are provided with a zero balance account upon account creation.

How can a customer change add Nominee details?

The user can route to the nominee details page and either choose to add or delete the nominee details by specifying the relation of the nominee to the account holder.

How can a customer change password?

The account holder can click on forgot password and fill in the details and proceed with the changing the password by entering a new one.

How can a user search for branch details?

The account holder choose either search the branch details by selecting the branch name or by entering the IFSC code.

How to obtain the transaction history?

The account holder can check the transaction history of the account by going to the transaction history page and entering the dates between which the transaction history needs to be obtained.
